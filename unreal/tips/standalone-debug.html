# Standalone 调试

Standalone 调试需要用 -game 启动 Unreal 项目。

# IDE 插件

  • Visual Studio : UnrealVS
  • Rider : EzArgs

安装并启用插件之后，会出现输入启动命令的文本框。

# 启动指令

"/Path/To/Your/Project.uproject" -game -windowed -log -verbose
